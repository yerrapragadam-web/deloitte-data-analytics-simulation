# Deloitte Australia - Data Analytics Job Simulation

This repository contains the deliverables completed during the Deloitte Australia Data Analytics Virtual Experience program on Forage. The simulation focused on data analytics, forensic technology, and dashboard creation to solve real-world corporate challenges.

## 🛠️ Project Overview

### Task 1 & 2: Daikibo Telemetry Analysis (Tableau)
* **Objective:** Analyze machine downtime logs from four international manufacturing facilities to identify operational bottlenecks.
* **Action:** Built a calculated field to track telemetry status logs and engineered an interactive Tableau dashboard. 
* **Insight:** Isolated **daikibo-factory-seiko** as the primary source of operational downtime, pinpointing specific **LaserWelder** equipment failures causing the bulk of productivity losses.

### Task 3: Corporate Equality & Compensation Audit (Excel)
* **Objective:** Audit employee compensation data across factory locations and job roles to flag equity issues.
* **Action:** Implemented absolute-value conditional formulas (`IF`, `ABS`) to parse score spreads ranging from -100 to +100.
* **Insight:** Classified data into Fair, Unfair, and Highly Discriminative categories to provide the legal and forensic teams an immediate map of pay gaps.
